# Blog Post Card

A simple and responsive Blog Post Card built using HTML and CSS. This project was created as part of my web development learning journey and is inspired by the FreeCodeCamp Responsive Web Design curriculum.

## Overview

The Blog Post Card displays a featured blog article with:

- A cover image
- Blog post title
- Short description (excerpt)
- "Read More" call-to-action link

The design focuses on clean layout, readability, and modern styling principles.

## Features

- Semantic HTML structure
- Responsive card layout
- Custom CSS styling
- Accessible image with descriptive alt text
- Hover effects for interactive elements
- Modern green color palette

## Technologies Used

- HTML5
- CSS3
