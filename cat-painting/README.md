# Cat Painting

A responsive CSS art project built with **HTML5** and **CSS3** as part of the **freeCodeCamp Responsive Web Design** curriculum. This project recreates a cat illustration using only HTML and CSS without relying on images, SVGs, or canvas.

## Features

- Pure HTML and CSS artwork
- Gradient-styled cat head
- Detailed ears, eyes, nose, mouth, and whiskers
- CSS shapes created with borders and border-radius
- Clean and organized HTML structure

## Technologies Used

- HTML5
- CSS3

## What I Learned

- Creating illustrations using only CSS
- Building shapes with borders and `border-radius`
- Positioning elements with CSS
- Applying gradients and transforms
- Layering elements with `z-index`
