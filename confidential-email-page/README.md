# Confidential Email Page

A simple HTML and CSS project that recreates a confidential email with classified stamps and blurred sensitive information.

## Features

- Confidential and Top Secret stamps
- Blurred sensitive text
- Custom styling with CSS transforms
- Centered email layout

## Technologies Used

- HTML5
- CSS3

## Project Inspiration

This project is inspired by the freeCodeCamp Confidential Email Page lab and focuses on practicing CSS filters, transforms, positioning, and styling.

## What I Learned

- Using CSS filters such as `blur()`
- Rotating elements with `transform`
- Styling and positioning content
- Creating realistic document layouts with CSS
