# Contact Form

A simple and responsive Contact Form built with HTML and CSS. Users can enter their name, email address, and a message before submitting the form.

## Features

- Name, email, and message fields
- HTML5 form validation with `required` attributes
- Clean and accessible form structure
- Responsive layout for different screen sizes
- Easy to customize and extend

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was created to practice building forms, working with form controls, and applying modern CSS styling techniques.

## Inspiration

Built as part of a web development learning journey inspired by freeCodeCamp projects and exercises.
