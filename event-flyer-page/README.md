# Event Flyer Page

A simple and responsive event flyer page built with **HTML** and **CSS** as part of the **freeCodeCamp** Responsive Web Design curriculum.

## Features

- Event banner with image and details
- Informative festival sections
- Two-column grid layout
- Responsive design
- Custom color palette and styling

## Technologies Used

- HTML5
- CSS3

## Project Inspiration

This project was inspired by the **freeCodeCamp Event Flyer Page Lab** and focuses on practicing semantic HTML, CSS Grid, spacing, typography, and visual design.

## What I Learned

- Structuring content with semantic HTML elements
- Creating layouts using CSS Grid
- Styling pages with colors, borders, and spacing
- Building visually appealing and responsive web pages
