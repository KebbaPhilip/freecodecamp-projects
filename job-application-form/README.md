# Job Application Form

A responsive and accessible Job Application Form built with HTML and CSS. The form allows applicants to enter their personal information, select a position, indicate their availability, and provide additional details about why they are interested in the role.

## Features

- Full name and email inputs
- Position selection dropdown
- Availability radio buttons
- Text area for additional information
- Form validation using HTML5 attributes
- Interactive styling with CSS pseudo-classes (`:focus`, `:valid`, `:invalid`, `:checked`, and `:first-of-type`)
- Responsive and user-friendly design

## Technologies Used

- HTML5
- CSS3

## Inspiration

This project was created as part of a web development learning journey and is inspired by form-building exercises from freeCodeCamp.
