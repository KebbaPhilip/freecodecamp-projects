# Newspaper Article

A simple newspaper article layout built with HTML and CSS as part of the freeCodeCamp Responsive Web Design curriculum.

## Features

- Newspaper-style article layout
- Headline and subheadline sections
- Author and publication date
- Styled article content
- Semantic HTML structure

## Technologies Used

- HTML5
- CSS3

## Project Inspiration

This project is inspired by the freeCodeCamp Newspaper Article lab and focuses on practicing semantic HTML, typography, spacing, and page layout techniques.

## What I Learned

- Using semantic HTML elements
- Structuring article content
- Working with typography and text styling
- Creating newspaper-inspired layouts
