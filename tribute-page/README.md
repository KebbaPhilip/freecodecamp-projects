# Tribute Page

A simple tribute page built with HTML and CSS as part of the freeCodeCamp Responsive Web Design Certification.

## Features

- Biography-style layout
- Image with caption section
- Timeline of key life events
- Blockquote for famous speech
- External link for more information
- Semantic HTML structure

## Technologies Used

- HTML5
- CSS3

## Project Inspiration

This project is inspired by the freeCodeCamp Tribute Page project. It focuses on practicing semantic HTML, typography, layout structure, and basic responsive design.

## What I Learned

- Using semantic HTML elements like `main`, `figure`, and `figcaption`
- Structuring long-form content
- Working with images and accessibility (`alt` text)
- Styling typography for readability
- Building a clean, centered page layout
